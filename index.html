<select is="location-select">0.1.0

The accessible, scrollable, styleable, horizontal carousel custom element.
No dependencies, about 12kB minified and gzipped.

Get started

Download the latest release:

github.com/stephband/location-select/releases

Include the JS and CSS files:

<link rel="stylesheet" href="./build/location-select.css" />
<script type="module" src="./build/location-select.js"></script>
